# Blutter Installer (Termux)

A simple and automated installer script to set up **Blutter** in Termux.

This tool prepares a complete environment for **Flutter reverse engineering** directly on Android devices without needing a PC.

---

## ⚙️ Features

- 📦 One-click installation
- 🔧 Auto installs all required dependencies
- 🐍 Python libraries setup
- 📁 Storage permission handling
- 🚀 Ready-to-use Blutter environment
- 📱 Fully mobile-friendly (Termux)
